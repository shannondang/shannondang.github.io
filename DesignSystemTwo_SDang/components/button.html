Button

Buttons are used to trigger a new page event in this design system, such as going to the "Work" page. There is only one button in this design system.

Primary Button
HTML
<button type="button" class="btn btn-style">Primary Button</button>
CSS
                  
                  .btn-style {
                    color: #C92127;
                    background-color: #F2EDE8;
                    border: 1px solid #C92127;
                    font-size: 16px;
                    padding: 8px 16px;
                    }
                  
                    .btn-style:hover {
                      background-color: rgb(201, 33, 39, .25);
                      border: 1px solid #C92127;
                      color: #C92127;
                    }

Accessibility Considerations

This button contains dark red text on a light cream background to meet the WCAG, Web Content Accessibility Guidelines, color contrast requirement. When hovered, the entire button turns a light red color, so users know that the mouse is near the button.
