Navigation Bar

The navigation bar faciliates browsing through the design system. The naviagtion bar is used for navigating through the pages of a site. Often times the current page is highligthed, underlined, so users know what page they are on.

Example
  • Home
  • About
  • Contact
HTML 
            
      <nav class="navbar navbar-expand-lg nav-bar-color sticky-top">
        <div class="container-fluid">
          <a class="navbar-brand" href="index.html"><img src="../logoimg/100w/logo.png" alt="primary logo"></a>
          <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNavDropdown"
            aria-controls="navbarNavDropdown" aria-expanded="false" aria-label="Toggle navigation">
            <span class="navbar-toggler-icon"></span>
          </button>
          <div class="collapse navbar-collapse" id="navbarNavDropdown">
            <ul class="navbar-nav ms-auto">
              <li class="nav-item">
                <a class="nav-link active nav-highlight" aria-current="page" href="#">Home</a>
              </li>
              <li class="nav-item">
                <a class="nav-link" href="about.html">About</a>
              </li>
              <li class="nav-item">
                <a class="nav-link" href="contact.html">Contact</a>
              </li>
            </ul>
          </div>
        </div>
      </nav>
CSS 
            
              .navbar{
                background-color: #F2EDE8;
                border-bottom: #C92127 1px solid ;
              }
              
              .nav-link{
                font-family: "Rufina";
                size:16px;
                color: #C92127;
              }
              
              .navbar-nav .nav-link.active{
                font-family: "Rufina";
                size:16px;
                color: #C92127;
                text-decoration: underline;
              }
              
              .navbar-nav > li{
                padding-left: 10px;
                padding-right: 10px;
              }

Accessibility Considerations

This navigation bar employs dark red text on a light cream background to meet the WCAG, Web Content Accessibility Guidelines, color contrast requirement. The logo has alternative text, so user who are blind can know a logo is there.
