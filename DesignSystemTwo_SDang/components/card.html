Accessibility Considerations

The card component has an alternative text for blind users. The hierchary of the headers shows the main ascepts of the card, which are project title and then the type of project. The "View Project" button contains dark red text on a light cream background to meet the WCAG, Web Content Accessibility Guidelines, color contrast requirement. When hovered, the entire button turns a light red color, so users know that the mouse is near the button.
